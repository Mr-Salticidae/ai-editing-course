## 六、必须避开的坑

1. **xfade/dissolve 会吃时长**——相邻两段各被 offset 吃掉一部分，config 里要预留补偿，否则音画错位
2. **PIL/ffmpeg drawtext 渲染中文必须指定字体绝对路径**（如 `C:/Windows/Fonts/msyh.ttc`），否则乱码
3. **素材文件名≠内容**——用图先看 `素材清单.md` 备注；`图标/` 子目录部分是带小红书水印低清网图，正式成片别用
4. **BGM 比片长短 49.8 秒**——循环补齐用交叉淡接，或尾段留白；BGM 压人声 18–22 dB
5. **绿幕素材**（`一键三连-绿幕.mp4`、`苹果手机框-绿幕.png`）需抠像（chromakey/colorkey），直接叠会带绿边
6. **大文件只在这台机器**——原片/录屏/BGM 不在 git 里，异机接手先确认文件存在
7. **P8 录屏切入点 2:08.3 起**——录屏本身 90s 比 P8 段（26s）长，需按方案选取片段，打字瞬间对齐 149.6s 的演示句
