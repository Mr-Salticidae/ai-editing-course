# 新手教程：.md 文件怎么打开

> 这个项目里的教程、说明、文案几乎全是 `.md` 结尾的文件（Markdown 格式）。它本质就是**纯文本**，但直接用记事本打开会看到一堆 `#` `*` 符号，不好读。下面教你怎么舒服地看。

## 什么是 .md 文件

Markdown 是一种"用符号标记格式"的纯文本：`#` 开头是标题、`-` 开头是列表、`**xx**` 是加粗。用对了工具，这些符号会自动变成排好版的样子（就像你现在看到的这页）。

## 方式一：直接在 GitHub 网页上看（零安装，最简单）✅

打开仓库 https://github.com/Mr-Salticidae/ai-editing-course ，点进任何 .md 文件，**GitHub 会自动排版渲染**，标题、表格、链接都是现成的。

- 优点：什么都不用装，手机也能看
- 缺点：只能看，不能改；本地文件夹里的文件看不了

**只是要"看教程"的话，用这个方式就够了。**

## 方式二：装 VS Code（本地看 + 编辑，推荐）✅

后面干活（改文案、写方案）需要在本地编辑，推荐 VS Code——免费、以后跟 AI 协作也用得上：

1. 官网下载：https://code.visualstudio.com/ （点大蓝按钮下载安装，一路下一步）
2. 用它打开 .md 文件（右键文件 → 打开方式 → VS Code）
3. 看到的还是带符号的原文？按 **`Ctrl + Shift + V`** 切换到"预览模式"，就是排好版的样子了
   - 也可以点右上角那个"分屏+放大镜"小图标，左边改、右边实时预览

## 方式三：Obsidian（喜欢笔记软件的选它）

免费的 Markdown 笔记软件：https://obsidian.md/ 。安装后"打开文件夹作为仓库"，选中项目文件夹，所有 .md 都能看能编辑，阅读体验最好。功能多，第一次打开可能觉得复杂，不强求。

## 常见问题

- **双击 .md 没反应 / 用记事本打开了一堆符号？** 正常，说明没关联阅读器。按方式二装 VS Code 后右键 → 打开方式 → 选 VS Code → 勾选"始终使用此应用"。
- **需要专门学 Markdown 语法吗？** 看教程不需要。以后要写东西时记住三条就够：`#` 空格后面写标题、`-` 空格后面写列表、空一行才算分段。
- **Word 能打开吗？** 能，但排版是乱的，别用 Word。
