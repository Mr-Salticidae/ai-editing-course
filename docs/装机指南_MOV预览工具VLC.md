# 装机指南：MOV 预览工具 VLC（绿色版）

> 为什么需要：`02_口播视频原稿/` 里的口播原片是 .mov 格式，Windows 自带的"电影和电视"经常打不开（HEVC/ProRes 等编码不支持）。VLC 免费开源，什么编码都能放。
> 推荐装**绿色版（zip 免安装）**：不写注册表、不占 C 盘，不要了直接删文件夹。2026-07-15 实测流程如下。

## 一、下载（官方源）

1. 打开官方目录：https://download.videolan.org/pub/videolan/vlc/last/win64/
2. 下载 **`vlc-x.x.x-win64.zip`**（约 76MB，注意是 `.zip` 不是 `.exe`）
   - Mac 用户去上一级目录找 `macosx/` 下的 `.dmg`，双击安装即可，后面步骤不用看

## 二、安装（其实就是解压）

1. 把 zip 解压到你想放的盘，比如 `D:\`
2. 解压出来的文件夹叫 `vlc-x.x.x`，改名为 `VLC`（可选，好认）
3. 完成。播放器就是 `D:\VLC\vlc.exe`，可以发送到桌面快捷方式

## 三、让 .mov 双击就能打开

第一次双击 .mov 文件时：

1. 右键 .mov 文件 → **打开方式** → **选择其他应用**
2. 列表里没有 VLC 的话，点"更多应用"→"在这台电脑上查找其他应用"，选 `D:\VLC\vlc.exe`
3. 勾选 **"始终使用此应用打开 .mov 文件"** → 确定

设一次，以后双击就直接开。

## 四、踩坑记录（想用官方 exe 安装器的先看这个）

我们最初用的是官方 `.exe` 安装器 + `/D=D:\VLC` 参数指定装 D 盘，结果**安装器两次无视参数装回 C 盘**：

- 第一坑：注册表 `HKLM\SOFTWARE\VideoLAN\VLC` 里残留的旧安装路径会覆盖命令行参数
- 第二坑：清了注册表它还是装回 C 盘默认路径，原因不明

结论：**不想折腾就直接用绿色版 zip**，效果完全一样。如果你无所谓装哪个盘，双击 exe 安装器一路下一步也行。

## 五、验证

双击打开 `02_口播视频原稿/` 里任意一个 .mov（或随便一个视频文件），画面声音正常即可。
