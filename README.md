## 四、交付物

1. **口播文案定稿**（分镜级：口播段落 + 对应画面提示）
2. **剪辑方案**（时间轴 / 卡点 / B-roll 插入点，蒙眼剪辑法产出）
3. **成片**（口播教学视频，含字幕）

## 五、产线（蒙眼剪辑法 6 步套用）

```
1. 口播文案定稿（主题图文先行）
2. 视频风格构思（教学片风格：口播为主 + 演示画面穿插）
3. 声音骨架（口播原声为骨架；BGM 垫底可选）
4. 剪辑节奏设计（AI 初标 B-roll/字幕落点，创作者审核）
5. 代码实现（ffmpeg / Remotion）
6. 反馈迭代
```
