## 你在这个项目里要做什么

一句话：**选一个你感兴趣的方向，为口播教学视频补充素材，交到 `03_剪辑物料/`。**

口播视频本身（录制、剪辑、成片）由跳蛛先生直接制作；你负责补充口播之外的素材，**方向待定、可自选**，比如：

- B-roll 画面（AI 生图/生视频、实拍、录屏演示均可）
- 图表 / 动画（讲解方法论用的示意图、流程图）
- BGM / 音效（可用 Suno）
- 其他你觉得对成片有帮助的东西

选定方向后知会跳蛛先生（开 Issue 或直接说）对齐一下即可动手。

学下面这套 AI 工作流仍然强烈推荐——做素材同样用得上，而且它本身就是这门课要教的内容。你不需要会剪辑软件，也不需要会写代码——核心就是：你跟 AI（Claude）对话当导演，AI 写代码干活，你负责看效果、提意见。

## 三步上手路线

### 第 1 步：装环境（约 20 分钟，一次性）

打开 [02_安装_照着做一遍.md](02_安装_照着做一遍.md)，Windows 用户直接双击 `一键安装_Windows.bat`（Mac 用 `一键安装_Mac.command`）。

可选：装一个能预览 .mov 的播放器（分工调整后你不经手口播原片了，但预览各类视频素材仍然好用）：照 [装机指南_MOV预览工具VLC.md](../docs/装机指南_MOV预览工具VLC.md) 做，5 分钟搞定。

### 第 2 步：明白原理（约 15 分钟阅读）

按顺序读两篇：
1. [00_先读我.md](00_先读我.md) — 这套东西是什么
2. [01_这套AI剪辑是怎么回事.md](01_这套AI剪辑是怎么回事.md) — 谁是总指挥、乐队里都有谁

然后跟着 [03_第一支片子_完整走一遍.md](03_第一支片子_完整走一遍.md) 练一支小片，里面有跟 AI 说话的话术卡。

### 第 3 步：干本项目的活

本项目是**口播教学视频**，成片由跳蛛先生制作；你按自选方向补充素材。乐队里的工具按你选的方向取用：

| 工具 | 什么方向会用到 | 说明 |
|---|---|---|
| Claude（总指挥） | ✅ 所有方向 | 构思、写 prompt、写代码做图表/动画都靠它 |
| Midjourney / Seedance（画面） | 补 B-roll 画面 | AI 生图/生视频素材 |
| Suno（作曲） | 补 BGM / 音效 | 片头/垫底 BGM |
| Remotion / ffmpeg | 补图表 / 动画 | 代码生成示意动画、流程图动效 |
| 字幕（WhisperX）/ ElevenLabs | ❌ 用不到 | 字幕和口播原声由跳蛛先生那边处理 |

**素材参考从哪看、产出交到哪：**
- 口播文案定稿：`01_口播文案/`（了解成片要讲什么，素材往上贴）
- 你产出的素材 → 提交到 `03_剪辑物料/`（大文件不入 git 的，走网盘/本地拷贝传递）
- 剪辑方案、成片：由跳蛛先生负责，不用你管

**剪辑方法**（了解即可）：蒙眼剪辑法 6 步（见仓库 [README.md](../README.md) 第五节）。

**技能已随仓库自带**：本项目要用的 4 个 AI 技能（蒙眼剪辑、歌曲字幕、Suno 配乐、Remotion）已装在仓库的 `.claude/skills/` 里——你用 Claude Code **打开这个仓库文件夹**干活，AI 就自动会这些本事，不用你手动装、也不用背。`skills/` 下的其他技能说明书同理，都是给 AI 读的。

## 卡住了怎么办

- 先把报错原文直接贴给 Claude，多数问题它能自己解决
- 解决不了的，在仓库开 Issue 或直接找跳蛛先生
- 每次干完活记得 `git pull` → 干活 → `git add`/`commit`/`push`，避免和别人改的冲突
